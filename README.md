 # 📈 Stock Market Anomaly Detection & Trend Forecasting

A data science project that detects anomalies in stock market data and forecasts future price trends using machine learning and time-series analysis.

## 🔍 Overview
- Fetches real-time stock data using **yfinance**
- Detects unusual price/volume behavior using **Isolation Forest**
- Forecasts next 7 days using **ARIMA**
- Interactive dashboard built with **Streamlit**

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (Isolation Forest)
- Statsmodels (ARIMA)
- Streamlit
- yfinance

## 📊 Features
- ✅ Real stock data fetching (any ticker)
- ✅ Anomaly detection on price & volume
- ✅ 7-day price forecasting
- ✅ Interactive Streamlit dashboard
